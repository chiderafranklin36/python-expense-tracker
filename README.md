# Python Expense Tracker

## Introduction
The Python Expense Tracker is a simple yet effective tool designed to help users manage and track their expenses efficiently. Built with Python, this application allows users to record their daily spending and categorize their expenses, providing insights into their financial habits.

## Features
- **Add Expense**: Record a new expense with details such as amount, category, and a note/description.
- **View Expenses**: Display all recorded expenses in a clear and organized manner.
- **Delete Expenses**: Remove any unwanted or incorrect expense entries.
- **View Total**: Instantly view your total spending to stay on top of your budget.

## Usage Instructions
1. **Clone the Repository**: 
   ```bash
   git clone https://github.com/chiderafranklin36/python-expense-tracker.git
   cd python-expense-tracker
   ```

2. **Run the Program**: 
   Execute the `expenses.py` file to start tracking your expenses.
   ```bash
   python expenses.py
   ```

3. **Interact with the Application**: 
   Follow the prompts in the terminal to add, view, delete, and check the total sum of your expenses.

## How to Run the Program
Make sure you have Python installed on your system (Python 3 recommended). Run:

```bash
python expenses.py
```

Follow the on-screen instructions to use all features.
